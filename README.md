# SMARCB1 Variant Analysis

## Project Overview

This project aims to investigate disease-associated SMARCB1 variants using computational biology and bioinformatics approaches.

## Research Question

Are pathogenic SMARCB1 variants preferentially located in evolutionarily conserved and structurally important regions of the protein?

## Objectives

1. Collect reported pathogenic SMARCB1 variants.
2. Analyze evolutionary conservation of affected residues.
3. Assess structural localization of variants.
4. Investigate potential functional consequences.

## Status

Project initiated June 2026.
